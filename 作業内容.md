## 技術的な詳細

### 使用技術
- HTML5
- CSS3（レスポンシブデザイン）
- JavaScript（Vanilla JS）
- LocalStorage API

### 主要な関数
- `addDeposit()`: 入金データの追加
- `addWithdraw()`: 出金データの追加
- `updateDepositTable()`: 入金テーブルの更新
- `updateWithdrawTable()`: 出金テーブルの更新
- `updateSummary()`: 収支サマリーの更新
- `getDayDepositTotal()`: 日別入金合計の取得
- `updateCalendar()`: カレンダーの更新
